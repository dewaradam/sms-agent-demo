# SMS Agent Demo · AI Agency Academy

A scrollable interactive demo showing how an SMS agent handles onboarding and ongoing tasks for service area businesses — no app, no portal, just text.

## What it shows

Eight modules across two phases:

**Onboarding**
- Platform Access (Leadsie tap-link)
- Service Area capture
- Business Hours
- Photo Upload

**Ongoing**
- Review Response (AI draft, owner approves)
- Social Posts (freeze warning, pipe tips)
- Review Requests (job receipt photo trigger)
- Listing / Search gap detection

## How to view it

**Live demo:** `https://[your-username].github.io/[repo-name]`

Or open `index.html` directly in any browser — no build step, no dependencies, no terminal needed.

## Built with

- React 18 (via CDN, no npm required)
- Babel standalone (JSX in browser)
- Pure inline styles, no CSS framework
